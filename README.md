# KickTracker

WoW addon that tracks interrupt cooldowns for your party/raid.

## Features
- Tracks all interrupt spells across all classes
- Syncs cooldowns between party/raid members running the addon
- Macro fallback for players without the addon (`/p kt:SPELLID`)
- Compact UI with class-colored bars, spell icons, and timers
- Settings menu via `/kt` — adjust scale, width, opacity, toggle features

## Install
1. Download the latest release zip from the [Releases](../../releases) page
2. Extract the `KickTracker` folder into `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or `/reload`

## Usage
- `/kt` — open settings menu
- `/kt dato` — send love in chat
